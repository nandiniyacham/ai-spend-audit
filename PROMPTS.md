# Prompt Strategy

The project uses AI-generated summaries to explain optimization opportunities.

Prompt goals:
- Explain savings clearly
- Keep tone startup-friendly
- Highlight unnecessary spending
- Encourage optimization actions

Current summary style:
- Concise
- Professional
- SaaS-focused
- Action-oriented

Future improvements:
- Personalized recommendations
- Industry-specific optimization advice
- Team-size-aware suggestions
- More advanced AI analysis
