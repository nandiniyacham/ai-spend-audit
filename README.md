# AI Spend Audit

A web application that audits startup AI tooling spend and recommends cost-saving optimizations across tools like ChatGPT, Claude, Cursor, GitHub Copilot, and Gemini.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- React
- UUID
- OpenAI SDK

## Features

- AI tooling spend audit form
- Savings recommendation engine
- Monthly and annual savings   calculations
- Shareable audit reports
- Persistent report storage with Supabase
- AI-generated audit summaries
- Lead capture flow
- Dynamic report pages
- Local storage persistence
