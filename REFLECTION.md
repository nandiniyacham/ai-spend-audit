# Reflection

This project improved understanding of:
- Next.js application architecture
- Supabase backend integration
- SaaS dashboard UX patterns
- Dynamic routing
- State management
- Database persistence

Biggest challenge:
Managing client/server separation while working with environment variables and AI integrations.

Most valuable learning:
Building a complete MVP workflow from frontend UI to persistent backend storage and dynamic reports.

What went well:
- Fast iteration speed
- Modular component structure
- Clear feature progression

What could improve:
- Better server-side AI integration
- More robust validation
- Enhanced analytics
